Справочник "Виды алкогольной продукции"

Справочник предназначен для хранения списка видов алкогольной и спиртосодержащей продукции.

Справочник используется при указании сведений об алкогольной продукции для целей заполнения декларации об объемах розничной продажи алкогольной и спиртосодержащей продукции.
