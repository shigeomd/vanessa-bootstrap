Сообщение об открытии (закрытии) счета, о вознкновении (прекращении) права использовать корпоративное электронное средство платежа для перевода электронных денежных средств (форма С-09-1)

Утверждена приказом ФНС России от 09.06.2011 г. № MMB-7-6/362@.

Для заполнения формы необходимо выбрать из прилагающихся списков либо заполнить:

  • Код признака типа организации.
  • Код признака события (открытие/закрытие счета, права использовать КЭСП).
  • Код признака места открытия/закрытия счета

В зависимости от выбора последних двух кодов дополнительно необходимо заполнить дополнительный Лист А, Лист Б либо Лист В. В случае открытия/закрытия счета в банке заполняется лист А. В случае заполнения сообщения об открытии/закрытии счета в органе федерального казначейства заполняется лист Б. В случае заполнения сообщения о возникновении/прекращении права использовать КЭСП заполняется лист В.

На листах А, Б, В необходимо заполнить необходимую информацию:

  • Номер счета.
  • Дата события.
  • Реквизиты банка.

По кнопке "Печать для ФНС" доступна печать сообщения по форме С-09-1.

По кнопке "Печать для ПФР" доступна печать сообщения об открытии/закрытии счета в банке (лист А) для ПФР.

По кнопке "Печать для ФСС" доступна печать сообщения об открытии/закрытии счета в банке (лист А) для ФСС.

По кнопке "Выгрузить" формируется электронное представление уведомления.
