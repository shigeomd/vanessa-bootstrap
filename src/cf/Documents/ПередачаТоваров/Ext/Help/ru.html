Документ "Передача товаров"

Документ предназначен для отражения операций по передаче собственных материалов стороннему переработчику для выпуска продукции и передачи заказчику продукции, выпущенной из давальческого сырья.

Передача сырья в переработку

Для отражения передачи сырья в переработку необходимо выбрать в поле Вид операции операцию Передача сырья в переработку. На закладке Товары нужно заполнить:

  • Счет передачи - счет учета материалов, переданных в переработку на сторону по каждому переработчику. С этом поле указывается субсчет счета 10.07 "Материалы, переданные в переработку на сторону".

Примечание
Для организаций, уплачивающих налог на прибыль, суммовая оценка прямых расходов, связанных со стоимостью материалов, переданных в переработку, определяется также, как в бухгалтерском учете, с учетом переработчиков (контрагентов).

Передача продукции заказчику

Для документального оформления передачи продукции заказчику (формирования печатной формы) необходимо выбрать в поле Вид операции операцию Передача продукции заказчику. Операция применяется только в том случае, когда передача продукции заказчику и оказание услуг заказчику по ее производству приходятся на разные моменты  времени. Если передача продукции заказчику и оказание услуг заказчику отражаются одновременно, то нужно регистрировать документ Реализация услуг по переработке.

Передача товаров комитенту

Для передачи товаров, закупленных в рамках договора комиссии (агентирования) комитенту (принципалу), в интересах которого был закуплен товар, используется вид документа Передача товаров комитенту. В поле Контрагент указывается комитент (принципал), в поле договор – договор комиссии (агентский договор) по закупкам. Табличная часть может быть заполнена автоматически по остаткам.

Печатные формы

Для документа Передача товаров предусмотрены следующие печатные формы:

  • М-11 (Накладная на отпуск материалов на сторону)
  • ТОРГ-12 (Товарная накладная)
  • 1-Т (Товарно-транспортная накладная)
