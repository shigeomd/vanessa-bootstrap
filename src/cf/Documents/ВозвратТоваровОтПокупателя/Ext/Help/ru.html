Документ "Возврат товаров от покупателя"

Документ предназначен для отражения возврата товаров, продукции, тары и оборудования от покупателя или от комиссионера. Документ можно ввести на основании документов: Реализация (акт, накладная), Отчет о розничных продажах.

Ввод на основании

При вводе документа в шапке обязательно нужно указать следующие реквизиты:

  • Склад - место хранения, на которое оформляется возврат. Возврат может оформляться только на склады с типом "Оптовый" или "Розничный".
  • Тип цен - цена товара, по которой оформляется возврат. Выбирается из нескольких допустимых для товара цен. Заполняется автоматически типом цен, который установлен в договоре покупателя или основным типом цен продажи, который установлен в настройках пользователя. Тип цен можно изменить по кнопке Редактировать цены и валюту. 
  • Валюта - валюта документа. Валюта заполняется автоматически валютой договора. 

Возврат товаров от покупателя с указанием документа реализации

Для отражения возврата товаров необходимо выбрать в поле Вид операции операцию Продажа, комиссия.

В шапке документа в поле Документ отгрузки нужно указать документ, которым была оформлена реализация возвращаемого товара.

После выбора операции нужно заполнить закладки:

  • Товары
  • Счета учета расчетов

На закладке Товары указывается номенклатура, количество, цена, ставка НДС, а также счета учета номенклатуры.

  • Цена - заполняется автоматически при указании номенклатуры на основании регистра Цены номенклатуры.
  • Счет учета, Счет учета НДС, Счет доходов и Счет расходов - заполняются автоматически при указании номенклатуры на основании регистра Счета учета номенклатуры.
  • Субконто - подставляется значение номенклатурной группы выбранного товара.

Для заполнения закладки Товары предназначена кнопка Заполнить:

  • Заполнить по документу отгрузки - закладка Товары заполняется данными из указанного в шапке документа реализации.

Если поле Документ отгрузки заполнено, то программа автоматически определит какова была учетная стоимость возвращаемых товаров в момент реализации и оприходует товары именно по этой учетной стоимости.

На закладке Расчеты заполняются счета расчетов с покупателем по возврату товара, а также, в случае если основанием возврата является документ "Отчет о розниных продажах" реквизиты приходного кассового ордера которым розничному покупателю были возвращениы денежные средства.

Возврат товаров от покупателя без указания документа реализации

Для отражения возврата товаров необходимо выбрать в поле Вид операции операцию Продажа, комиссия.

После выбора операции нужно заполнить закладки:

  • Товары
  • Счета учета расчетов

На закладке Товары указывается номенклатура, количество, цена, ставка НДС, а также счета учета номенклатуры.

Если поле Документ отгрузки не заполнено, необходимо дополнительно указать учетную стоимость возвращаемых товаров.

  • Поле Себестоимость предназначено для указания учетной стоимости возвращаемых товаров. В этом поле указывается общая сумма, по которой товары будут оприходованы в бухгалтерском учете. Себестоимость всегда указывается в рублях и не зависит от валюты документы.
  • Цена - заполняется автоматически при указании номенклатуры на основании регистра Цены номенклатуры.
  • Счет учета, Счет учета НДС, Счет доходов и Счет расходов - заполняются автоматически при указании номенклатуры на основании регистра Счета учета номенклатуры.
  • Субконто - подставляется значение номенклатурной группы выбранного товара.

Для заполнения закладки Товары предназначена кнопка Заполнить:

  • Заполнить по документу отгрузки - позволяет выбрать документ реализации, данные из которого будут перенесены на закладку Товары, при этом ранее введенные строки будут очищены.
  • Добавить из документа отгрузки - позволяет выбрать документ реализации, данные из которого будут добавлены на закладку Товары в дополнение к уже существующим строкам.

Примечание
При упрощенной системе налогообложения, если поле Документ отгрузки не заполнено, нужно дополнительно заполнить реквизит Расходы (НУ). Нужно указать, отнесены ли возвращаемые товары в момент реализации к принимаемым расходам.

Возврат товаров от комиссионера

Для отражения передачи товаров или готовой продукции комиссионеру необходимо выбрать в поле Вид операции операцию Продажа, комиссия. При этом в поле Договор должен быть указан договор, имеющий вид "С комиссионером (агентом)". 

После выбора операции нужно заполнить закладки:

  • Товары

  • Счета учета расчетов

На закладке Товары указывается номенклатура, количество и счета учета номенклатуры.

  • Счет учета и Переданные счет учета - заполняются автоматически при указании номенклатуры на основании регистра Счета учета номенклатуры.

Возврат оборудования

Для отражения возврата оборудования необходимо выбрать в поле Вид операции операцию Оборудование.

После выбора операции нужно заполнить закладки:

  • Оборудование
  • Счета учета расчетов

На закладке Оборудование указывается номенклатура, количество, цена, ставка НДС, а также счета учета номенклатуры.

Если поле Документ отгрузки не заполнено, необходимо дополнительно указать учетную стоимость возвращаемого оборудования.

  • Поле Себестоимость предназначено для указания учетной стоимости возвращаемого оборудования. В этом поле указывается общая сумма, по которой оборудование будет оприходовано в бухгалтерском учете. Себестоимость всегда указывается в рублях и не зависит от валюты документа.
  • Цена - заполняется автоматически при указании номенклатуры на основании регистра Цены номенклатуры.
  • Счет учета, Счет учета НДС, Счет доходов и Счет расходов - заполняются автоматически при указании номенклатуры на основании регистра Счета учета номенклатуры.

Для заполнения закладки Оборудование предназначена кнопка Заполнить:

  • Заполнить по документу отгрузки - позволяет выбрать документ реализации, данные из которого будут перенесены на закладку Оборудование, при этом ранее введенные строки будут очищены.
  • Добавить из документа отгрузки - позволяет выбрать документ реализации, данные из которого будут добавлены на закладку Оборудование в дополнение к уже существующим строкам.

Возврат тары

Для отражения возврата тары необходимо выбрать в поле Вид операции операцию Продажа, комиссия или Оборудование (если тара возвращается вместе с оборудованием).

После выбора операции нужно заполнить закладки:

  • Тара
  • Счета учета расчетов

На закладке Тара указывается номенклатура, количество, цена и счета учета тары.

  • Цена - заполняется автоматически при указании номенклатуры на основании регистра Цены номенклатуры.
  • Счет учета - заполняется автоматически при указании номенклатуры на основании регистра Счета учета номенклатуры.

Для заполнения закладки Тара предназначена кнопка Заполнить:

  • Заполнить по документу отгрузки - позволяет выбрать документ реализации, данные из которого будут перенесены на закладку Тара, при этом ранее введенные строки будут очищены.
  • Добавить из документа отгрузки - позволяет выбрать документ реализации, данные из которого будут добавлены на закладку Тара в дополнение к уже существующим строкам.

Печатные формы

Для документа Возврат товаров от покупателя предусмотрены печатная форма:

  • Возврат от покупателя

Ввод на основании

На основании документа Возврат товаров от покупателя можно ввести следующие объекты:

  • Отражение начисления НДС
  • Отражение НДС к вычету
  • Платежное поручение
  • Выдача наличных
  • Списание с расчетного счета
  • Счет-Фактура выданный
  • Счет-Фактура полученный
