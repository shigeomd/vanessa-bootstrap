Можно ли из программы передать контрагенту произвольный файл с ЭЦП?

Для этой цели можно использовать "Произвольный электронный документ", который является аналогом электронного письма и позволяет заполнить сопроводительную записку, вложить необходимый файл, установить ЭЦП (если требуется), и направить документ в адрес контрагента.
Документ можно использовать для обмена неформализованными документами, например, для передачи договора или акта сверки.
Хранение версий для произвольного электронного документа не поддерживается.
