Предназначен для работы с файлами, присоединенными к документу.

Открывается по команде Присоединенные файлы () командной панели документа.

В списке выводится:

  • Наименование - полное имя файла. Значок рядом с наименованием указывает тип файла.

 

С помощью команды Еще - Дополнительно - Изменить форму можно добавить сведения:

  • Описание - подробные сведения о файле.

Добавление нового файла

  • Новый файл можно Добавить из уже имеющегося файла на диске.

Просмотр файла

  • Графические файлы отображаются в области предварительного просмотра. Для просмотра во внешней программе дважды кликните по наименованию файла в списке.

Передача файлов по электронной почте

  • Для отправки присоединенных файлов по электронной почте выделите требующиеся файлы, удерживая кнопку shift, и нажмите кнопку Отправить... или воспользуйтеся сообветствующей командой контекстного меню.

Просмотр карточки файла

  • Нажмите Еще - Открыть карточку (нажмите клавишу F2), для того чтобы просмотреть свойства выделенного в списке файла. Также можно воспользоваться соответствующей командой контекстного меню.

Сохранение файла

  • С помощью команды Еще - Сохранить как можно сохранить файл в указанный каталог на жестком или сетевом диске.
  • Также можно воспользоваться соответствующей командой контекстного меню по правой кнопке мыши.
  • Соответствующей командой меню Еще или контекстного меню по правой кнопке мыши можно Открыть папку файла в Проводнике Windows.
