Предназначена для проверки легальности получения обновлений.

При обновлении программа запрашивает у администратора подтверждение, что файлы обновления были получены легальным способом.

Выберите с помощью переключателя:

  • Я подтверждаю легальность получения обновления в соответствии с вышеизложенными условиями - в этом случае будет проведено обновление программы.
  • Я НЕ подтверждаю легальность получения обновления в соответствии с вышеизложенными условиями - в этом случае работа программы будет завершена.
