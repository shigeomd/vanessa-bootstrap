Таблица 6-2 "Регистр прочих расходов, связанных с осуществлением предпринимательской деятельности" используется для расшифровки прочих расходов. В графе "Вид расходов" для затрат по социальному страхованию указывается наименование счета затрат, для других расходов указывается наименование статьи затрат.
