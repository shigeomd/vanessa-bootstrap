Доступно формирование печатной формы договора и приложения к нему.
Для этого необходимо нажать кнопку Печать - Договор, выбрать один из шаблонов для заполнения. После чего текст договора можно распечатать, сохранить в присоединенном файле или отправить по электронной почте.
По кнопке Печать - Приложение к договору можно подготовить приложение  с перечнем товаров, услуг.
