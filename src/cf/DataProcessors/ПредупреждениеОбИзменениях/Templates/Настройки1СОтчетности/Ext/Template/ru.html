Мастер подключения к 1С-Отчетности перенесен в командную панель (кнопка "Подключение к 1С-Отчетности"). Расширенные настройки подключения доступны через: Отчеты -> Регламентированные отчеты -> Настройки.

Ссылка на физическое лицо для ИП больше не показывается в форме. Редактирование дополнительных личных данных ИП возможно через: Справочники -> Физические лица.

По умолчанию скрыты поля:
  • Регистрационный номер в ТФОМС.
    Отобразить их можно с помощью команды "Еще - Изменить форму".
