По ссылке Документы можно быстро перейти к списку документов по договору. 
С помощью ссылки Присоединенные файлы отображается список хранимых по договору файлов (скан-копии и т.д).

Доступно формирование печатной формы договора.
Для этого необходимо нажать кнопку Договор, выбрать один из шаблонов для заполнения. После чего текст договора можно распечатать, сохранить в присоединенном файле или отправить по электронной почте.

По умолчанию скрыты поля:

  • Код;
  • Входит в группу;
  • НДС по ставкам 4% и 2%.
    Отобразить их можно с помощью команды "Еще - Изменить форму".
