Предназначена для печати формы согласия на обработку персональных данных.

Федеральный закон от 27.06.2006 № 152-Ф3 "О персональных данных" предписывает со всех субъектов, чьи персональные данные в программе обрабатываются, получить согласие на такую обработку.

Открывается из списков программы или из элементов списков по команде Еще - Согласие на обработку персональных данных. Печать формы возможна из всех списков программы, в которых имеются персональные данные.

Оператор персональных данных

  • Заполните Дату согласия.
  • В поле Организация выберите из списка наименование организации - оператора персональных данных.
  • Поле Адрес заполняется автоматически, если оно заполнено в списке Организации. Если оно не заполнено, можно заполнить его вручную или (при наличии соответствующих прав) заполнить адрес в карточке Организации и выбрать ее из списка повторно, для того чтобы поле Адрес заполнилось автоматически.
  • В поле Ответственный за обработку ПДн напишите ФИО сотрудника, который вводит личные данные.

При этом возможна ситуация, когда данные субъектов уже были внесены в программу (до вступления в силу закона, до принятия в компании положения и т.д.), а возможна ситуация, когда данные субъекта пока не внесены. То есть после утверждения положения, принимаем сотрудника, и до ввода его данных в программу формируем бланк согласия. Для этого в программе поддерживаются два режима печати формы согласия.

Печать формы для новых сотрудников

  • В случае если сотрудник принимается на работу, его персональные данные пока не введены в программу. Кадровик формирует печатную форму согласия, в которой заполнены данные организации, но не заполнены данные физического лица, вместо них нижнее подчеркивание для заполнения. Сотрудник заполняет свои данные, подписывает документ, после этого кадровик вводит данные в программу, согласие остается в организации в бумажном виде.
  • Если данные о субъекте на момент составления согласия в программе еще отсутствуют (сотрудник только принимается на работу), то их можно указать непосредственно в печатной форме или распечатать форму с графами для заполнения, для этого переключатель Персональные данные субъекта необходимо переключить в положение пока отсутствуют в информационной базе.

Печать формы с использованием данных программы

  • В случае если персональные данные субъектов уже существуют в программе, кадровик выбирает из списка тех субъектов персональных данных, от которых нужно получить согласие, выводит печатные формы согласия. При этом в форме будут заполнены все данные, как организации, так и субъектов персональных данных.
  • Для печати формы Согласие на обработку персональных данных на каждого субъекта персональных данных необходимо в соответствующей строке списка или в одном из элементов списка воспользоваться командой Еще - Согласие на обработку персональных данных.
  • Для того чтобы распечатать форму согласия для физических лиц, данные о которых есть в программе, переключатель Персональные данные субъекта переключите в положение будут использованы из информационной базы, при этом имеющиеся в программе данные по выделенным в списке субъектам будут помещены в табличную часть.
  • Для печати формы на несколько элементов списка (например, сотрудников одной организации) выделите несколько записей с помощью левой кнопки мыши и клавиш Ctrl или Shift.
  • Если было выделено несколько записей списка, то их данными заполняется табличная часть документа. Можно уточнить недостающие в таблице данные по каждому субъекту (Адрес и Паспортные данные).
  • При необходимости внесите в список недостающие записи с помощью кнопки Добавить, напишите данные вручную.
  • Если список в табличной части содержит несколько человек, то в этом случае количество сформированных печатных форм будет равно количеству строк в табличной части (для каждого элемента списка будет сформирована своя форма).

Печать формы согласия

  • Для печати формы в виде табличных документов формата .mxl нажмите кнопку Печать, далее укажите нужное количество копий, нажмите кнопку Печать.
  • Для печати в формате Microcoft Word или OpenOffice.org Writer используйте соответствующие кнопки. Необходимо иметь в виду, что на компьютере должно быть установлено соответствующее приложение.
